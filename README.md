# 🤖 Nova – AI Powered Discord Study Assistant

Nova is an AI-powered Discord bot built using **Spring Boot**, **JDA**, **Groq AI**, and **MySQL** that helps students learn more effectively without leaving Discord.

It provides AI-powered explanations, interactive quizzes, flashcards, code debugging, study roadmaps, XP-based gamification, and moderation tools—all in one place.


---
## ✨ Features
### 🎓 AI Learning Assistant
- 💬 Ask study-related questions
- 📖 AI-generated explanations with examples
- 💻 Code explanation
- 🐞 Code debugging assistance
- 🗺️ Personalized study roadmaps
- 🔄 Text-based flowchart generation
- 📄 PDF summarization
### 📝 Quiz System
- AI-generated multiple-choice quizzes
- Interactive Discord buttons
- Score tracking
- XP rewards
- Automatic level progression
### 📚 Flashcards
- Generate AI flashcards
- Save flashcards
- Review saved flashcards anytime
### 👤 User System
- User registration
- XP & Level system
- Personal profile
- Leaderboard rankings
### 🛡️ AI Moderation
- Detects toxic or spam messages
- Sends moderation reports
- Keeps servers cleaner

# 🚀 Commands
| Command | Description |
|----------|-------------|
| `/ping` | Check whether Nova is online |
| `/register` | Register your Discord account |
| `/profile` | View your profile, XP & level |
| `/ask` | Ask any study-related question |
| `/quiz` | Start an interactive AI quiz |
| `/flashcards` | Generate AI flashcards |
| `/myflashcards` | View saved flashcards |
| `/review` | Review saved flashcards |
| `/leaderboard` | View top users |
| `/explaincode` | Explain any code snippet |
| `/debug` | Debug code using AI |
| `/roadmap` | Generate a study roadmap |
| `/flowchart` | Generate a text flowchart |
| `/summarizepdf` | Summarize an uploaded PDF document |
| `/serverinfo` | Display Discord server information |

# 🔒 Security
Sensitive information is never stored inside the repository.
Nova uses environment variables for:
- Discord Bot Token
- Groq API Key
- Database Credentials
Example:
```properties
discord.token=${DISCORD_TOKEN}
groq.api.key=${GROQ_API_KEY}
spring.datasource.url=${DB_URL}
spring.datasource.username=${DB_USERNAME}
spring.datasource.password=${DB_PASSWORD}
```
